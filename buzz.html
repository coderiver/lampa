Instagram добавит кнопку «Купить сейчас» для страниц компаний

После полутора лет экспериментов с форматами и таргетингом для страниц брендов, Instagram решил ввести кнопки «Купить сейчас», «Установить сейчас» и «Зарегистрироваться», которые позволять пользователям взаимодействовать с компаниями.

Ещё соцсеть добавит таргетинг по интересам и демографии аудитории и API для управления крупными кампаниями.

Издатель Fallout запустил таинственный обратный отсчёт на своём сайте

2 июня на сайте студии Bethesda появился обратный отсчёт с заставкой «Пожалуйста, ждите», известной по культовой серии игр Fallout. Судя по счётчику, он должен закончиться в 7 часов вечера 3 июня по московскому времени. Как предполагают профильные СМИ, на конференции E3 студия собирается представить игру Fallout 4.
